# Sustainability Impact Logger

## Overview

A cloud-ready web application for recording sustainability-related software engineering decisions.

## Tech Stack

- Next.js
- TypeScript
- PostgreSQL
- Prisma
- Docker

## Features

- Create impact items
- View impact items
- REST API
- PostgreSQL persistence
- Health endpoint

## Architecture

User
→ Next.js
→ Prisma
→ PostgreSQL
